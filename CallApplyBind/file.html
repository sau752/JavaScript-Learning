Call Apply Bind Basics

The one similarity between all of these functions is that their first argument is always the 'this' value, or context

Call Function - Call is the simplest of them all. It is exactly the same as invoking a function as you would do normally, however the difference is that you also get to specify the functions context.


            var person1 = {
                firstName: "Saurabh",
                lastName: "Verma"
            };

            function callFunction(greetingMessage) {
                alert(`${greetingMessage}, ${this.firstName} ${this.lastName}`);
            }

            $("#anchorCallFunction").on("click", function () {
                callFunction.call(person1, "Hello");
            });

Apply Function - Apply is exactly the same as call apart from the fact that you pass in the functions arguments as an array and not separately.



            var person2 = {
                firstName: "Milind",
                lastName: "Verma"
            }

            var greetMessage = ["Good", "Morning"];

            function applyFunction(firstMsg, secondMsg) {
                alert(`${firstMsg} ${secondMsg}, ${this.firstName} ${this.lastName}`);
            }

            $("#anchorApplyFunction").on("click", function () {
                applyFunction.apply(person2, greetMessage);
            });

Bind Function - The bind method enables you to pass arguments to a function without invoking it. Instead, it returns a new function with the arguments bound preceding any further arguments.


            function bindFunction(greetingMessage, middleName) {
                alert(`${greetingMessage}, ${this.firstName} ${middleName} ${this.lastName}`);
            }

            $("#anchorBindFunction").on("click", function () {
                var callFun = bindFunction.bind(person1, "Hello");
                callFun("Blank");
            });
